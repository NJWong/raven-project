## List all arm units {{ tag: 'GET', label: '/api/v1/units/arm' }}

<Row>
  <Col>

    Returns a paginated list of all arm units.

    ### Optional query parameters

    <Properties>
      <Property name="limit" type="integer">
        Pagniation limit for units. Defaults to 20 if not specified.
      </Property>
      <Property name="offset" type="integer">
        Pagniation offset for units. Defaults to 0 if not specified.
      </Property>
    </Properties>

    <div className="not-prose">
      <Button
        href="/pagination"
        variant="text"
        arrow="right"
        children="See Pagination Guide for more information"
      />

  </Col>


